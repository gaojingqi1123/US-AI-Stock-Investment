## 文件1：《为未来撒下种子》
**日期：2025年12月2日**

### 1. 核心投资观点或理念
- **"先做确定性强的事，先拿到确定性强的结果"** — 市场无法预测，当目标进入击球点时就先击球，之后再根据市场走势拿到自己想要的结果。
- **"万物皆周期，任何的人和事，都不会一直在云端"** — 在春风得意时布局，平淡了才有退路。
- **"只在胜率和赔率都对自己极度有利时下重注，其他时间保持耐心"** — 这是不频繁买卖却能取得好成绩的核心。
- **"钱是手段，而不是目的，人才是目的"** — 做好经济保障，让人摆脱焦虑，获得时间自由，这才是财富的意义。
- 绝大多数人靠运气赚的钱，应该放到不靠运气也能赚钱的地方（构筑防守型和稳健型资产）。
- 认识到自己是个普通人，认识到万物皆周期，认识到运气造就人的成功——做到这三点，成就低不到哪去。

### 2. 具体的投资策略/操作方法
- **"第一兼唯一"选股原则**：选择拥有强大护城河的公司，逢跌分批买入，逢高分批卖出。
- **金字塔加减仓法**：英伟达110、100、95、86美元分批买入，越低买得越多；175、180、185、190、195、200美元时分批合计卖出30%。
- **凯利公式思维框架**：
  - 在有优势时下注，没有优势时观望（f*为负或零时不下注）→ 几乎不追高疯涨个股
  - 优势越大（高胜率、高赔率），下注可以越重，但永远不要押上全部
  - 风险（波动）是下注的敌人，波动越大越让人恐惧，因此做低成本/负成本和构筑防守型资产很有必要
- **"半凯利"策略**：计算出的f*再除以2，牺牲少量增长率的同时大幅降低资金曲线波动和破产风险。
- **账户分类配置**：进取型、稳健型、防守型账户资金的配比符合"最优下注比例"。

### 3. 对市场走势的判断
- 市场无法预测，不做频繁预测，专注于做好眼前的事。
- 99.9%的人连过去发生了什么都搞不清楚，更别说预测未来了。

### 4. 实际操作记录
- 英伟达：110、100、95、86美元分批买入；175-200美元分批合计卖出30%
- 伯克希尔A：持有五六年，只在77.5万美元时减仓过1股，其余长期持有；伯克希尔B持有至今没减仓
- 近期持仓调整：个股精减、不同风格账户占比调整、防守型和稳健型账户重点增配
- 白银、伦铜：准备再拿一段时间看看
- 香港保险：围绕离岸信托之外配置保单

### 5. 风控思路
- **追求极高胜率**：只投资自己能看懂的企业、不懂不做、"第一兼唯一"、长期持有为主
- **追求极高赔率**：在别人恐惧时贪婪（市场恐慌性抛售是提供高赔率的最佳时机）；用4毛钱买价值1元的东西；控制好安全边际
- **风险控制**：看懂公司、用闲钱投资、不满仓、慎用杠杆、留足现金、构筑好防守型资产和进取型资产的占比，整体风格稳健为主
- 纯粹进取型甚至赌徒式投机的人，资产体量往往较小；当资产体量到一定程度，防守型/稳健型资产存在的必要性就体现出来

### 6. 关注的板块/品种
- 个股：英伟达、伯克希尔
- 贵金属：白银、伦铜
- 港险配置

### 7. 重要的人生感悟或投资哲学
- **"万物皆周期，任何的人和事，都不会一直在云端。在春风得意时布局，平淡了才有退路。"**
- **"钱是手段，而不是目的，人才是目的。"**
- **"真正珍贵的终点，是深厚而有意义的人际关系。心要像伞一样，撑得起，收得住，望可望之事，待可待之人。"**
- **"幸福是我们来到这世上应该追求的，它应该是一场持续且漫长的旅程，偶尔有波动，但应是持续累加，而不是持续减少。"**
- **"手里的金钱，只是保持自由的一种工具；我们追求富有、地位、知识和健康，也只不过是为了不被金钱、权势、愚昧和疾病所束缚。追求的，是自由。"**
- 专注于自身，多从自己身上找问题并解决，能让自己成长飞速。人到中年，更应如此。
- 有些人自己菜鸡一个，却喜欢通过贬低他人来抬高自己——山脚的野草长不高，也会抱怨被山顶的大树挡住了阳光和雨水。
- 经济压力会直接影响家庭健康，应该稳步致富，越来越富，和人际关系的升温、递进一样。

## 文件3：《调仓，长梦已醒》
**日期：2025年11月26日**

### 1. 核心投资观点或理念
- **"未来并非我所能预见，但我始终相信'科技是第一生产力'，相信科技改变未来。"**
- 配置重心一直都在科技股上，所买的个股大多数符合"第一兼唯一"的特征。
- **"成功者先相信后看见，普通人先看见后相信，失败者看见了都不信。"**
- 这世上哪有什么永恒不变的。草原上的动物尚且知道季节迁移，往有水、草的地方走，作为高级哺乳动物的人，有些却还是习惯于"守株待兔、刻舟求剑"。
- 始终让自己保持学习能力，不断精进，才是应对这个不断变化的世界最好、最安全的举措。

### 2. 具体的投资策略/操作方法
- **三个账户重新调整定位**：
  - **进取型账户**：美股七巨头+台积电、博通、AMD，以及CRCL等彩票股。剥离两个宽基指数ETF转移到稳健型账户。
  - **稳健型账户**：最大体量持仓纳指100和标普500两个宽基指数ETF（50%+）；消费板块20%+（Costco、沃尔玛、麦当劳）；医药保健板块29%（礼来、联合健康、强生）。把科技股转移到进取型，可口可乐和伯克希尔移到防守型。
  - **防守型账户**：主要是美债和相关ETF（BIL和TLT）占比最大，其次是SCHD、伯克希尔、可口可乐、MO、强生等，还有大量现金储备。
- **目标配比**：从约5:2:3调整为4:1.5:4.5
- **四种卖出情况**：(1)做T和小幅度减底仓控安全边际；(2)PEG≥2时卖出；(3)有更好投资目标时切换；(4)公司基本面恶化时。
- **"先相信，后看见"的受益逻辑**：用输得起的钱去试错，用输不起的钱保底。10次试错里有7-8次失败，但2-3次成功带来的回报是失败损失的10倍甚至100倍。

### 3. 对市场走势的判断
- 美股未来数周预计将持续震荡
- 目前处于人工智能浪潮中前期往中期过渡的阶段
- 前期受益股：英伟达、博通、AMD等芯片"铲子股"；微软、亚马逊、谷歌等云计算巨头；台积电（全流程受益）

## 跨文件核心主题总结

### 一、投资体系核心框架
1. **"第一兼唯一"选股原则**：只投拥有强大护城河的公司
2. **三大账户配置**：进取型（科技股）:稳健型（宽基ETF+消费医药）:防守型（美债+高股息+现金）= 4:1.5:4.5
3. **金字塔加减仓法**：逢跌分批买入（越低买越多），逢高分批卖出
4. **凯利公式思维**：只在胜率和赔率极度有利时下重注，其他时间保持耐心
5. **四种卖出情形**：做T控安全边际 / PEG≥2 / 更好目标切换 / 基本面恶化

### 二、核心风控原则
1. **不满仓、不上杠杆、不追高**
2. **用输得起的钱试错，用输不起的钱保底**
3. **用闲钱投资、留足现金、慎用杠杆**
4. **先做确定性强的事，先拿到确定性强的结果**
5. **在别人恐惧时贪婪，用4毛钱买价值1元的东西**
6. **万物皆周期，任何人和事都不会一直在云端**

### 三、市场判断要点
1. AI浪潮：从中前期往中期过渡，前期受益芯片股，中期受益应用端和云业务
2. 谷歌：AI大战中必有其一席之地，是科技巨头中的"伯克希尔"
3. 英伟达：GPU是唯一具备通用平台能力的产品，地位稳固

5. 美股整体偏贵，当前以防守为主
